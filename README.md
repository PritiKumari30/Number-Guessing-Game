
# 🎯 Number Guessing Game

A simple interactive number guessing game built using HTML, CSS, and JavaScript.  
The player has 10 attempts to guess a randomly generated number between 1 and 100.

## 🚀 Features
- Random number generation (1–100)
- High / Low hints
- Attempt counter (10 tries)
- Previous guesses display
- Restart game option

## 🛠️ Tech Stack
HTML | CSS | JavaScript (Vanilla JS)

## 📂 Project Files
index.html  
style.css  
script.js
